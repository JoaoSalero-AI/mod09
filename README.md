# Observation Methodology in Youth Football — GitHub Pages Site

This repository is a **static website** (HTML/CSS/JS) intended to be published with **GitHub Pages**.

## What’s inside
- `index.html` — Overview
- `microcycle.html` — Weekly microcycle calendar
- `sessions.html` — PSS/SSP sessions with diagrams
- `individual.html` — Individual session + report template
- `video-feedback.html` — Recording setup + feedback workflow
- `print.html` — Print-ready single-page view
- `assets/diagrams/` — SVG diagrams used across pages
- `styles.css`, `app.js` — Styling and small interactions (theme toggle, copy buttons)
